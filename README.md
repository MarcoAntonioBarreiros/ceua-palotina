# CEUA/Palotina - GitHub Pages

Pagina estatica da Comissao de Etica no Uso de Animais da UFPR - Setor Palotina.

Link publico: https://marcoantoniobarreiros.github.io/ceua-palotina/

## Como atualizar datas, membros e documentos

Edite somente o arquivo `dados-ceua.js`.

No GitHub:

1. Abra `dados-ceua.js`.
2. Clique no icone de lapis.
3. Altere somente os valores entre aspas.
4. Mantenha virgulas, colchetes e chaves.
5. Clique em `Commit changes`.

O que pode ser atualizado nesse arquivo:

- `reunioes`: datas das reunioes ordinarias.
- `composicao`: portaria, subtitulo e membros titulares/suplentes.
- `coordenacao`: nome e e-mail da coordenacao.
- `documentos`: links de regimento, resolucoes, formularios e cursos.

O arquivo principal da pagina e `index.html`, mas ele nao precisa ser alterado para atualizacoes rotineiras.
